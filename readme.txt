Fast customizable jmpress plugin for wordpress

== Description ==


This plugin developed for easily set up a page and conver it to slideshow with using jmpress.js (http://jmpressjs.github.io/jmpress.js/#/home)

All sub pages of selected page will be slide of its.

Easily set up for "Inline Attributes" (http://jmpressjs.github.io/docs/inline.html)


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload \`mingusjmpress\` to the \`/wp-content/plugins/\` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place \`[mingusjmpress]\` shortcode in any page that you want to convert to be slideshow.
